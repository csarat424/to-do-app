# Simple To-Do App

This is a simple to-do app built with HTML, CSS, JavaScript, and Node.js. The app allows you to add tasks, mark them as completed, and features a Dockerfile for easy deployment on an EC2 Ubuntu Instance

## Features

- Add tasks to your to-do list.
- Mark tasks as completed with a click.
- Centered design with a pleasant background.
- Dockerized for easy deployment.

## Getting Started

### Prerequisites

- Node.js: Make sure you have Node.js installed on your local machine. You can download it from [nodejs.org](https://nodejs.org/).

- Git: Make sure you have Git installed on your local machine.
